# BGP Integration

Route advertisements, no-overlay routing, and EVPN support via BGP.

**[Route Advertisements](route-advertisements.md)**

Advertise pod and service CIDRs to external BGP peers for direct routing.

**[No-Overlay Mode](no-overlay.md)**

Route pod traffic directly using BGP-learned routes without tunnel encapsulation.

**[EVPN](evpn.md)**

Scalable multi-tenant networking using Ethernet VPN with BGP control plane.
